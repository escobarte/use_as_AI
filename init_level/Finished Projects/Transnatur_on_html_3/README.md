# Transnatur INC — Landing Page Project

## Quick Start

1. Open Claude Code
2. Run: `claude`
3. Paste this prompt:

```
Read SKILL.md in this folder and build the full Transnatur INC recruitment 
landing page exactly as described. Create index.html, form.html, and 
thankyou.html. Use all branding colors and images from the assets/ folder.
```

| File | Description |

| `SKILL.md` | Full project documentation — branding, structure, content, Google Sheets setup |
| `assets/Ttransnatur_logo_white.png` | Logo for dark backgrounds |
| `assets/Ttransnatur_logo_black.png` | Logo for light backgrounds |
